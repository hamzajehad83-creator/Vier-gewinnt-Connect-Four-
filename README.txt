# Connect Four (Vier gewinnt)

A JavaFX implementation of the classic Connect Four game.

## Features
- Two-player gameplay (Red vs Green)
- Interactive column buttons
- Win detection (horizontal, vertical, diagonal)
- Help dialog with Wikipedia link
- New game reset functionality

## How to Run
1. Ensure Java 11+ and JavaFX are installed
2. Compile: `javac --module-path /path/to/javafx --add-modules javafx.controls,javafx.web application/*.java`
3. Run: `java --module-path /path/to/javafx --add-modules javafx.controls,javafx.web application.Main`
